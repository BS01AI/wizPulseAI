## 🎯 使用原则

1. **品牌名称**：WizPulseAI 保持英文，不翻译
2. **产品名称**：QuickSlide, DataHive, FlowBridge 等保持英文
3. **术语优先级**：日语自然表达 > 英语直译
4. **专业术语**：AI、API、Dashboard、OAuth 等保持英文或片假名
5. **一致性原则**：同一概念在所有站点必须使用相同术语
6. **用户友好**：选择用户最熟悉的表达方式

## 📝 特殊说明

### 登录术语选择

**为什么选择「ログイン/Login」而不是「サインイン/Sign In」？**

1. **更广泛使用**：日本网站和应用普遍使用「ログイン」
2. **用户习惯**：用户更熟悉「ログイン」这个词
3. **一致性**：与其他主流平台保持一致
4. **简洁性**：「ログイン」更简短、更直观

### 注册术语保持

**为什么「サインアップ/Sign Up」保持不变？**

1. **行业标准**：「サインアップ」是注册的通用术语
2. **无更好替代**：日语中没有更自然的表达
3. **国际通用**：Sign Up 在全球范围内被广泛理解

## 🔄 术语演进历史

### 2025-11-12 修正

**修改内容**：
- 统一"登录"术语（サインイン → ログイン）
- 统一"登录"英文（Sign In → Login）

**修改原因**：
- Auth站点与Main/Dashboard站点术语不一致
- 影响用户跨站点体验

**影响范围**：
- Auth站点：`src/lib/i18n/auth-center.ts`
- 翻译文档：创建 `TRANSLATION_GLOSSARY.md`
